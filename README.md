# 🎉AviUtl『メモ帳』プラグイン

* [AviUtl](http://spring-fragrance.mints.ne.jp/aviutl/)のプラグインです。
* メモ帳として使える複数行エディットボックスを表示します。

## ⬇️ダウンロード

* [最新バージョンはこちら](https://github.com/hebiiro/memo.auf/releases/latest/)

## 🚀導入方法

1. pluginsフォルダをaviutlフォルダにコピーします。

## 💡使い方

1. AviUtlのメニューから『表示』➡『メモ帳の表示』を選択します。
1. 『メモ帳』ウィンドウが表示されます。
1. 『メモ帳』ウィンドウのエディットボックスにメモを書き込みます。

## 🔧設定方法

* 『メモ帳』ウィンドウのメニューから設定を変更できます。

### 📝設定項目

* `プロジェクトと連動`
	* ✏️これにチェックが入っている状態でプロジェクトを保存すると、プロジェクトファイルにメモが埋め込まれて保存されます。
	* ✏️これにチェックが入っている状態でプロジェクトを読み込むと、プロジェクトファイルに埋め込まれているメモを読み込みます。
